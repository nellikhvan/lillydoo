Lillydoo test application
========================

Create an an address book CRUD.

**The address book should contain the following data:** 
- Firstname
- Lastname
- Street and number
- Zip
- City
- Country
- Phonenumber
- Birthday
- Email address
- Picture (optional)

**The following techniques were used:** 
- Symfony 3.4
- Doctrine with SQLite
- Twig
- PHP7.4+
